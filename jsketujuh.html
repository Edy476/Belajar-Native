Array

apa itu array?

Array adalah tipe objek yang dirancang untuk menyimpan kumpulan data.
Karakteristik utama dari array JavaScript adalah:

  • Element : Sebuah array adalah daftar nilai, yang dikenal sebagai elemen.
  • Terurut : Elemen array diurutkan berdasarkan indeksnya.
  • Berbasis Indeks 0 : Elemen pertama berada pada indeks 0, elemen kedua pada indeks 1, dan seterusnya.
  • Ukuran Dinamis : Array dapat bertambah besar atau menyusut seiring penambahan atau penghapusan elemen.
  • Heterogen : Array dapat menyimpan elemen dengan tipe data yang berbeda (angka, string, objek, dan array lainnya).

Contoh Membuat Array

const array_name = [item1, item2, ...];
const cars = ["Saab", "Volvo", "BMW"];


Bisa juga buat kosong dulu, seperti dibawah ini
const cars = [];
cars[0]= "Saab";
cars[1]= "Volvo";
cars[2]= "BMW";


Menggunakan kata kunci JavaScript `new`
const cars = new Array("Saab", "Volvo", "BMW");


Cara mengakses array
const cars = ["Saab", "Volvo", "BMW"];
document.getElementById("demo").innerHTML = cars[0];


Akses Seluruh Array
const cars = ["Saab", "Volvo", "BMW"];
document.getElementById("demo").innerHTML = cars;


Cara mengubah elemen array
const cars = ["Saab", "Volvo", "BMW"];
cars[0] = "Opel";
document.getElementById("demo").innerHTML = cars;
